# PUC MINAS

Repositório com as atividades e documentos de aula de minha graduação na PUC MINAS. Atualmente, curso Engenharia de Software, mas entrei na universidade pelo curso de Sistemas de Informação. Os vídeos são muito pesados para subir pelo GitHub, limitando-me a enviar para o Drive. E todos os documentos foram compactados e transformados em PDF (caso o original não fosse).

Como portadora de diploma, algumas disciplinas fui dispensada, mas cada disciplina foi organizada por pastas, com o nome da disciplina. 

:exclamation: A pasta `_infos-puc` são as matrizes curriculares, grade curricular e os requisitos de cada disciplina.
